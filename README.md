# Docking — Projet BlueROV (Guerlédan)

Ce dépôt est la plateforme commune pour partager le code et le rapport du projet Docking de Guerlédan — objectif : faire entrer un BlueROV dans une cage en s'appuyant uniquement sur un sonar.

## Contenu du dépôt

- `ros2_bluerov/` — Workspace ROS 2 principal (ex « ros2_ws ») : simulation du sonar, chaîne de perception, mission complète et pilotage à la manette — ouvrir le dossier [ros2_bluerov](ros2_bluerov).
- `simulation/` — Simulation 2D du BlueROV pour tester et comparer les lois de commande — ouvrir le dossier [simulation](simulation).
- `docs/` — Ressources documentation (images pour les READMEs) et sources LaTeX du rapport (docs/Rapport_Docking/) — ouvrir le dossier [docs](docs/).
- `divers/` — Codes obsolètes/expérimentaux archivés (exemples et scripts de test) — ouvrir le dossier [divers](divers/).
